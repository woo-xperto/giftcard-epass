Integrate Epasscard with WooCommerce Gift Cards to allow seamless gift card purchases and redemptions using Epasscard.

== Description ==

The **Epasscard to Gift Card Extension** bridges the gap between the [WooCommerce Gift Cards](https://woocommerce.com/products/gift-cards/) plugin and the Epasscard payment system. This plugin enables customers to purchase and redeem gift cards using their Epasscard balance, offering a smoother and more flexible checkout experience.

**Key Features:**

- Seamless integration between Epasscard and WooCommerce Gift Cards.
- Allows customers to redeem gift cards using Epasscard.
- Supports gift card purchases via Epasscard payment method.
- Lightweight and easy to configure.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/epasscard-giftcard-extension` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Ensure both the **WooCommerce Gift Cards** plugin and the **Epasscard** plugin are installed and activated.
4. Configure Epasscard settings under WooCommerce > Settings > Payments.
5. You're ready to go!

== Frequently Asked Questions ==

= Does this plugin work with any gift card plugin? =  
No. This extension is specifically built to work with the official **WooCommerce Gift Cards** plugin.

= Is Epasscard required for this plugin to work? =  
Yes, you must have the Epasscard plugin installed and configured for this extension to function properly.

= Can customers use both gift cards and Epasscard in a single transaction? =  
Yes, depending on your WooCommerce and Epasscard settings, customers can combine payment methods.
